# 🌟 Team Showcase - Landing

Una landing page minimalista para presentar a los integrantes de un equipo.


## 🗂️ Estructura del Proyecto

```
/
├── index.html                          # Página principal (Home)
├── main.css                            # Estilos principales
├── src/
│   ├── pages/
│   │   ├── team-collection-page/       # Página con todos los miembros
│   │   │   ├── team-collection-page.html
│   │   │   └── team-collection-page.css
│   │   └── team-member-page/           # Página individual de miembro
│   │       ├── team-member-page.html
│   │       └── team-member-page.css
│   └── assets/
│       ├── img/                        # Imágenes del proyecto
│       └── icons/                      # Iconos
├── .gitignore
└── README.md
```

## 🚀 Cómo Usar

1. **Abrir el proyecto:**
   ```bash
   # Simplemente abre index.html en tu navegador
   open index.html
   # O con live server en VS Code
   ```

## ️ Tecnologías

- HTML5 (semántico)
- CSS3 (Grid, Custom Properties, Gradients)
- Sin frameworks ni librerías
- Sin Node.js ni build tools

## 📱 Responsive Breakpoints

- **Desktop**: > 768px
- **Mobile**: ≤ 768px
